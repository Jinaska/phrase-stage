# PHRASE STAGE 💜

英熟語 Vintage 1051–1113(+その他の重要イディオム、全80フレーズ)を楽しく覚えるための学習Webアプリ。

## モード

- 🇬🇧→🇯🇵 英語 → 日本語 4択クイズ
- 🇯🇵→🇬🇧 日本語 → 英語 4択クイズ
- ⌨️ タイピングモード(A/Bは省略OK、大文字小文字も気にしなくてOK)
- 📖🔊 フレーズリスト(英語の読み上げ音声つき・検索対応)

## お楽しみ要素

コンボを重ねると…🔥→💜 Borahae →🕺 Oppa Gangnam Style →🧨 DYNAMITE!!
スコアに応じて「練習生」から「Billboard No.1級」までランクが決まるよ。

## 使い方

`index.html` をブラウザで開くだけ。GitHub Pages でもそのまま動きます。
